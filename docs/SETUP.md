# ContractGuard AI - Setup Guide

Complete setup instructions for ContractGuard AI.

## Prerequisites

- **AWS Account** with Bedrock access enabled
- **Python 3.11+**
- **Node.js 18+** (for AWS CDK)
- **AWS CLI** configured with credentials
- **Git**
